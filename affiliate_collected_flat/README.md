# Affiliate Collected Flat

Thư mục này là bản gom phẳng từ các nguồn affiliate đã chọn lọc. Cấu trúc chỉ giữ 2 nhóm:

- `images/`: ảnh sản phẩm, ảnh demo, ảnh dùng cho bài đăng
- `copywriting/`: caption, kế hoạch nội dung, link, report, checklist, tài liệu triển khai

## Nên mở trước

1. `copywriting/Shopee_Affiliate_INSTAGRAM_CAPTION.md`
2. `copywriting/Shopee_Affiliate_MAU_BAI_DANG.md`
3. `copywriting/Shopee_Affiliate_LINK_AFFILIATE_5_MON.md`
4. `copywriting/Product_Research_product_research.md`

## Quy ước tên file

- Tiền tố `Shopee_Affiliate_` và `Product_Research_` giữ lại để biết nguồn gốc.
- Ảnh và tài liệu gốc không bị sửa nội dung, chỉ đổi sang cấu trúc phẳng để dễ tìm.

## Mục đích sử dụng

- Chọn sản phẩm
- Lấy caption / script / CTA
- Ghép ảnh vào bài Instagram, Facebook, Threads, TikTok
- Tái sử dụng nhanh cho các đợt đăng tiếp theo
