THƯ MỤC ẢNH NHÂN VẬT
====================

Bỏ ảnh nhân vật của bạn vào đây với tên:  player.png
-> Đường dẫn đầy đủ: sprites/player.png

Lưu ý:
- Nên là ảnh PNG nền trong suốt (transparent).
- Ảnh pixel art kích thước nhỏ (vd 16x16, 16x24, 32x32) trông đẹp nhất, nét nhất.
- File tên khác (vd hero.png) hoặc .jpg -> đổi tên thành player.png,
  hoặc báo Claude để đổi đường dẫn trong code.

Chưa có ảnh thì game vẫn chạy (hiện khối pixel cam tạm).
